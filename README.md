## 简介

GrapWork 是一款强大的跨平台桌面 AI Agent 助手，支持任何 OpenAI 兼容的 LLM API，提供丰富的 AI 生产力功能。

## 核心功能

- **多模型支持** - 兼容任意 OpenAI 格式 API（OpenAI、Claude、DeepSeek、通义千问等）
- **多标签对话** - 标签式会话管理，支持搜索功能
- **流式响应** - 实时流式输出，支持多种推理格式
- **MCP 工具调用** - Model Context Protocol 集成，支持 STDIO/SSE 传输
- **全局记忆** - 持久化知识存储，智能关键词匹配
- **技能系统** - 基于 SKILL.md 的领域知识包
- **图片生成** - 多提供商支持（智谱 GLM-Image、通义万相）
- **工作空间** - 内置文件浏览器和管理功能
- **隐私安全** - 所有数据本地存储

## 使用案例
