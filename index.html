Core Features

核心功能

简洁高效的 AI 对话体验，集成多种强大功能

多标签对话

支持多会话标签管理，流式响应实时显示，可配置温度、Top-P 等对话参数

MCP 工具调用

支持 STDIO/SSE 两种传输方式，自动发现并执行 MCP 服务器提供的工具

全局记忆

持久化知识存储，支持关键词智能匹配，自动注入相关上下文到对话中

技能系统

基于 SKILL.md 的领域知识包，支持多优先级来源，按需加载注入

社区助理

创建自定义 AI 助手，配置专属角色和 System Prompt，一键切换场景

工作空间

内置文件浏览器，支持目录浏览、文件预览、搜索等操作

图片生成

多提供商支持（智谱 GLM-Image、通义万相），会话历史独立管理

Markdown 渲染

完整 Markdown 解析，代码语法高亮，可切换代码主题

推理过程展示

支持 DeepSeek、Qwen 等推理模型，可展开查看完整思考过程

灵活配置

支持多个 LLM API 配置，兼容 OpenAI、Claude、本地模型等

跨平台支持

基于 Electron 构建，支持 macOS、Windows 和 Linux

隐私安全

所有数据本地存储，不上传云端，可插拔存储后端
